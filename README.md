## Storybook

Clone repository

```
git clone https://github.com/jloo87/NZMSA-Phase2.git
```

Install dependencies

```
npm install
```

Start storybook


```
npm run storybook
```


Open on browser with URL: [http://localhost:6006/](http://localhost:6006/)
